Учебный проект (веб-страница) "Евклид"

Сайт компании строительной компании Евклид.
Немного о работе:
- страница открывается и корректно работает на любых устройствах с разрешением от 1920px до 320px
- все классы реализованы по методогии БЭМ
- верстка велась по PixelPerfect (максимальное отклонение от макета - 3px)
- все одинаковых элементов созданы общие классы (нет ненужного нагромождения кода)
- кастомизирован чекбокс в footer
- применены различные JavaScript библиотеки (Swiper(секция "Hero"), Accordion (секция "Часто задаваемые вопросы"), InputMask (секция "Контакты")) 
- применён чистый JavaScript (бургер-меню, табы в секции "Как мы работаем")
