Дипломная работа по курсу "Веб-вёрстка".

Сайт компании Blanchard.
Немного о работе:
- страница открывается и корректно работает на любых устройствах с разрешением от 1920px до 320px
- все классы реализованы по методогии БЭМ
- все одинаковых элементов созданы общие классы (нет ненужного нагромождения кода)
- применены различные JavaScript библиотеки (Swiper(в секциях "Hero", "Галерея", "События", "Проекты"), Accordion (секция "Каталог"), Choices (секция "Галерея"), Tippy (секция "Проекты"), InputMask (секция "Контакты"), JustValidate (секция "Контакты")) 
- применён чистый JavaScript (бургер-меню, табы в секции "Каталог")
- добавлены Яндекс.карты с геолокацией офиса компании 


С данной веб-страницей можно ознакомиться по ссылке: http://test-blanchard.webtm.ru/
